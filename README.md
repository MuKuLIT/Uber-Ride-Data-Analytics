# 🚖 Uber Data Analysis Dashboard (Power BI)

## 📊 Project Overview
This project is a **Power BI dashboard** designed to analyze Uber ride data, focusing on **rider behavior, revenue trends, vehicle performance, and trip outcomes**.

It provides actionable insights into **bookings, cancellations, customer trends, and location-based demand** to support data-driven decision making.

---

## 🚀 Key Insights
- 📈 **92.6K total bookings** with **51.8M total revenue**
- ❌ High **lost bookings (56.8K)** indicating demand-supply gaps  
- 🚗 Revenue breakdown across vehicle types (Auto, Bike, Sedan, XL)
- ⭐ Customer rating: **4.4 | Driver rating: 4.2**
- 📍 Top locations: **Khandsa (pickup), Ashram (drop)**
- 💳 Most used payment method: **UPI**

---

## 💡 Business Value
- Identify **revenue drivers and high-performing vehicle types**
- Reduce **ride cancellations & lost bookings**
- Optimize **fleet allocation by location and time**
- Improve **customer experience through insights**


## 📌 Features
- **Interactive Dashboard Navigation**
  - Overview, Vehicle, Revenue, Rider, Location 
- **Booking Analysis**
  - Completed, Cancelled, and Incomplete rides  
- **Revenue Trends**
  - Monthly & quarterly performance tracking  
- **Customer Insights**
  - First-time vs returning riders  
- **Location Intelligence**
  - High-demand pickup/drop zones  
- **Time Slot Analysis**
  - Peak booking hours across weekdays
